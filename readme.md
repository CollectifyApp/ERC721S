## About The Project

ERC721S, **S for SoulBound**, is an implementation for SoulBound (Non-transferable) NFT coined by Vitalik Buterin in a [blog post](https://vitalik.ca/general/2022/01/26/soulbound.html).

## Features

- Non-transferable
- Community recovery
- Gas saving
- Sybil labelling
- Soul power
