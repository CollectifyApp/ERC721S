## About The Project

ERC721S, or ERC721 SoulBound, is an implementation for SoulBound Token (SBT) coined by Vitalik Buterin in a [blog post](https://vitalik.ca/general/2022/01/26/soulbound.html).

## Features

- Non-transferable
- Community recovery
- Gas saving
- Sybil labelling
- Soul power
